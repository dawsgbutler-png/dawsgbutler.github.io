3. Click Repository > Show in Explorer.
4. Keep the hidden .git folder. Do not delete it.
5. Delete or replace the old website files inside the repository folder.
6. Copy EVERYTHING INSIDE this Version 2 folder into the repository folder.
   index.html must be at the top level next to styles.css.
7. Return to GitHub Desktop.
8. Summary: Launch portfolio version 2
9. Click Commit to main.
10. Click Push origin.
11. Wait one to three minutes and refresh the live site with Ctrl+F5.
