DAWSON BUTLER DIGITAL ENGINEERING PORTFOLIO

STATUS
This is a complete, responsive static portfolio website. It includes:
- Home, About, Projects, Experience, Skills, Certifications, and Contact sections
- Seven individual engineering case-study pages
- Dark and light modes
- Mobile navigation
- Scroll animations with reduced-motion support
- Resume and capstone downloads
- Search/social metadata, favicon, and optimized WebP images
- No external libraries, trackers, or paid services

PREVIEW LOCALLY
1. Unzip the folder.
2. Open index.html in a browser.
3. All pages and downloads work offline.

PUBLISH WITH A PUBLIC URL
GitHub Pages:
1. Create a new public repository, such as dawson-portfolio.
2. Upload the contents of this folder, not the outer folder.
3. Open Settings > Pages.
4. Set Source to Deploy from a branch, choose main and /root, then save.
5. GitHub will provide a public URL.

Netlify:
1. Sign in to Netlify.
2. Open the manual deploy or drag-and-drop page.
3. Drag this entire folder into the upload area.
4. Netlify will immediately create a public URL.
5. Rename the site in Site configuration for a cleaner address.

CUSTOM DOMAIN
After publishing, buy a domain such as dawsonbutler.com or dawsonbutlerengineering.com and connect it through the host's domain settings.

BEFORE MAKING IT PUBLIC
- Confirm that Gaylor and LUCID Bots imagery is approved for public portfolio use.
- Do not add client names, proprietary dimensions, drawing numbers, or internal documents.
- The trading dashboard is labeled as a paper-trading interface concept and does not claim real investment performance.

EDITING
Text is in index.html and the files inside the projects folder.
Global design is controlled by styles.css.
Theme, mobile menu, and animations are controlled by script.js.
